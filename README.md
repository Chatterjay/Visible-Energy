# Visible Energy

一个 FluxNetworks 的 NeoForge 附属模组，在游戏世界中高亮显示通量网络设备的位置、传输速率和网络拓扑。

## 功能

- **设备高亮** — 使用通量网络对应颜色的半透明色块和线框标记扫描区域内的所有 Flux 设备
- **连接线** — 同一网络内的输入点（PLUG）到其他设备之间绘制彩色细线，直观展示网络拓扑
- **信息标签** — 每个设备顶部显示名称、网络名称、能量传输速率和网络占比百分比
- **动态刷新** — 每 1 秒自动更新数据，实时反映能量变化
- **跨维度监控** — 检测并提示网络中位于其他维度的设备，显示坐标和区块加载状态
- **彩色提示** — 绿色表示能量输入，红色表示高占比输出

## 指令

| 指令 | 说明 |
|------|------|
| `/ve` | 使用默认半径扫描周围 Flux 设备 |
| `/ve scan [radius]` | 使用指定半径（5~128）扫描 |
| `/ve stop` | 停止当前扫描 |
| `/ve duration <seconds>` | 设置高亮显示时长（1~3600 秒） |

## 配置

通过 `visible_energy-common.toml` 配置：

- `scanRadius` — 扫描半径，默认 32，范围 5~128
- `displayDuration` — 高亮显示时长（秒），默认 30，范围 5~300

## 依赖

- Minecraft 1.21.1
- NeoForge 21.1.234+
- FluxNetworks 8.0.0+
