# Visible Energy

FluxNetworks 附属模组，在游戏世界中高亮显示通量网络设备的位置、传输速率和网络拓扑。

## 功能

- **设备高亮** — 用网络对应颜色的半透明色块和线框标记扫描区域内的所有 Flux 设备
- **连接线** — 同一网络内从输入点（PLUG）到其他设备绘制彩色连接线
- **信息标签** — 每个设备上方显示名称、网络名、传输速率、占比和网络总储能
- **动态刷新** — 每秒自动更新，实时反映能量变化
- **防抖平滑** — EMA 平滑比例显示，避免数值跳动
- **跨维度监控** — 检测网络中其他维度的设备，显示坐标和加载状态
- **配置界面** — 游戏内 GUI 配置，无需编辑文件

## 指令

| 指令 | 说明 |
|------|------|
| `/ve` | 用默认半径扫描周围设备 |
| `/ve scan [radius]` | 指定半径（5~256）扫描 |
| `/ve stop` | 停止动态扫描 |
| `/ve clear` | 立即清除所有高亮 |
| `/ve duration <seconds>` | 设置高亮时长（1~3600 秒） |

别名：`/visible_energy`

## 配置

游戏内 Mods → Visible Energy → Config 可配置：

| 分类 | 选项 | 默认 | 范围 |
|------|------|------|------|
| 显示 | 高亮时长（秒） | 30 | 5~600 |
| 防抖 | 平滑系数 | 0.4 | 0.0~1.0 |
| 防抖 | 更新阈值 | 3 | 0~100 |
| 防抖 | 最大丢失容忍 | 3 | 1~10 |
| 扫描 | 扫描半径 | 64 | 5~256 |
